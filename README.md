# ga-satellite-deorbit

This repository is a unity project that simulates the optimization of satellite deorbiting using genetic algorithms. The project is a part of the course "ゲームAI・人工生命" at the University of Tokyo.

## How to run the project

1. Clone the repository
Run `git clone git@github.com:ryomappi/ga-satellite-deorbit.git` in your terminal or just download the zip file.

2. Open the project in Unity
Open Unity Hub and click on "Add" to add the project folder.
Then click on the project to open it.

3. Run the project
There are two scenes in the project. The first scene "GA" is the main scene where the genetic algorithm is implemented. The second scene "TestBestGene" is a test scene where you can test the best gene found in the main scene. To run the project, you should fill out "Best Gene Path" in "Tester" object in the scene.
